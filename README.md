# kusalakari.org

Static web pages generator for kusalakari.org

## Development

```bash
$ git clone https://github.com/ninegene/kusalakari.org.git
$ sudo npm install -g harp
$ cd kusalakari.org
$ ./run-dev-server.sh
```

Browse to `http://localhost:9999/`

## Deploy

```bash
$ ./deploy.sh
```
